About Us

"GreenTea" is an application designed to know noise levels of the BKC campus. Using data, that is stores, Users can know which area of the BKC campus has the least and most noise disturbance. This app is leaning towards to students or workers who wants peace and quiet when working

Application Description

Noise Level Map


Simple Description
Study location finder application which would provide noise level information in public spaces within the Ritsumeikan University BKC campus.
Data Collection
Users would be able to measure noise levels inside buildings using a smartphone phone noise measurement application. This noise level data can be uploaded with the time interval data and location/building name. Processes such as averaging would be taken place on this data before being displayed on the interactive map.
Application Design
An interactive map of the BKC campus would be shown with a noise level meter above public locations indicating the noise level of that specific location. The noise level meter would also be color coded so the user can understand the general noise level without requiring the understanding of the noise level measurements. A time slider would also be included so the user can read the information at different times. At this moment the time periods would be in 90-minute intervals.
Purpose
Through this application, users who seek a quiet space to study on the campus can look for quiet locations for studying or taking Zoom lessons. The user can easily read the average noise levels for the user's desired time frame, which would give the user an understanding of the general noise levels of different locations on the campus.
    Further development possibilities
  • We could expand the location for this application to a larger area. So the whole city's common locations can be displayed, which would give much more purpose to this application to the general public.
  • A shorter time interval can be used so the time accuracy would be increased.
  • A longer time frame can be used for users who are interested in knowing the noise levels at a later, uncommon time.

    • Functional Requirements:
  • The application shall collect and store noise level data along with corresponding time intervals and location/building names.
  • The application shall perform data processing, such as averaging, on the uploaded noise level data.
  • The application shall display an interactive map of the Ritsumeikan University BKC campus.
  • The map shall indicate the noise level of each public location on the campus using a noise level meter.
  • The noise level meter shall be color-coded to provide a visual representation of the general noise level at each location.
  • The application shall include a time slider to allow users to view noise level information for different time periods.
  • The application shall provide users with the average noise levels for their desired time frames (90-minute intervals).
  • The application shall help users identify quiet locations for studying or taking Zoom lessons on campus.

    • Non-functional Requirements:
  • The application shall have a UI (user interface) that is easy to understand and navigate.
  • The application shall reliably collect and store noise level data without data loss.
  • The application shall handle a large volume of noise level data and display it efficiently on the map.
  • The application shall be compatible with popular smartphone platforms (e.g., iOS and Android).
  • The application shall support various screen sizes and resolutions for optimal user experience.

    • Future Development Possibilities:
  • The application should have the potential to expand its coverage beyond the Ritsumeikan University BKC campus to include common locations in the entire city.
  • The application should allow for shorter time intervals to increase the time accuracy of noise level measurements.
  • The application should provide options for users to view noise level information for longer time frames, especially for uncommon times of the day or night.
